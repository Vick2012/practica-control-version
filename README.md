# Práctica de Control de Versiones

Esta práctica forma parte del laboratorio de **Estándares y Métricas de Calidad de Software**, enfocada en el uso de Git y GitHub para la gestión de configuraciones.

## Autor
Victor Alfonso Alarcón Ospina

## Objetivo
Aplicar buenas prácticas de control de versiones mediante ramas, commits, pull requests y resolución de conflictos.

## Estándares aplicados
- ISO/IEC/IEEE 12207:2017 (Gestión de configuraciones)
- ISO 9001:2015 (Sistema de Gestión de Calidad)
